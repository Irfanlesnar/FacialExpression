Work on Spark AR

Create Shapekeys

Create shapekeys or blendshapes using your favourite software. and export the model as (.gltf) or (.fbx)

 Should not export it as .obj because it does not contains the shapekeys or bones or any animation in the exported file.

Importing into Spark AR

Import the file inside the spark AR studio. and select the object which contains shapekeys. In my case the object is face. so when I select the face. On the right side panel all the shapekeys will be shown. If you cant see any shapekeys on right side panel the you should export in any other format. Mostly (.gltf) will work. If it doesnt work you can try (.fbx). If you are using Blender then I recommend to export it as (.gltf or .glb).

Get Into patch editor

Once you found the shapekeys. click on the yellow arrow button near the shapekeys panel. Now the value of the shapekeys can be changed in real-time.

Connecting with patch

After clicking the little arrow. you can see a yellow box inside the patch editor. just connect the patch to the yellow box. Similarly connect your shapekeys to the respective patch.
